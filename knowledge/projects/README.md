# Проекты AVsound

Каждый проект или мероприятие — отдельный `.md` файл.

## Формат файла проекта

```markdown
---
title: Название мероприятия
date: YYYY-MM-DD
client: Имя клиента
status: планируется | подготовка | выполнено | отменено
tags: [свадьба, корпоратив, концерт]
created: YYYY-MM-DD
---

# Название мероприятия

**Дата:** YYYY-MM-DD  
**Клиент:** [ссылка на клиента]  
**Статус:** планируется

## Оборудование

- [ ] [позиция 1]
- [ ] [позиция 2]

## Задачи

- [ ] [задача]

## Заметки

[заметки]
```

## Как добавить проект

Скажи Claude: **"запомни проект: [название] на [дата] для [клиент]"**
