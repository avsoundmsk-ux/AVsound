# Клиенты AVsound

Каждый клиент — отдельный `.md` файл. Имя файла = имя клиента.

## Формат файла клиента

```markdown
---
name: Иван Иванов
phone: +7 999 000 00 00
email: ivan@example.com
type: физлицо
tags: [свадьба, корпоратив]
created: YYYY-MM-DD
---

# Иван Иванов

**Телефон:** +7 999 000 00 00  
**Email:** ivan@example.com

## История заказов

- YYYY-MM-DD: [описание заказа]

## Заметки

[Любые заметки о клиенте]
```

## Как добавить клиента

Скажи Claude: **"добавь клиента [Имя] [телефон]"**
